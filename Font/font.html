If font is specified as a system keyword, it must be one of:

  1. caption: The system font used for captioned controls (e.g., buttons, drop-downs, etc.).
  2. icon: The system font used to label icons.
  3. menu: The system font used in menus (e.g., dropdown menus and menu lists).
  4. message-box: The system font used in dialog boxes.
  5. small-caption: The system font used for labeling small controls.
  6. status-bar: The system font used in window status bars.

If font is specified as a shorthand for several font-related properties, then:
it must include values for:

  1. font-size
  2. font-family
  3. font-variant
  4. font-weight
  5. font-strecth
  6. font-height
  • font-style, font-variant and font-weight must precede font-size
  • font-variant may only specify the values defined in CSS 2.1, that is normal and small-caps
  • font-stretch may only be a single keyword value.
  • line-height must immediately follow font-size, preceded by "/", like this: "16px/3"
  • font-family must be the last value specified.
