2. Box alignment in CSS Grid Layout

  • The place-content property is shorthand for align-content and justify-content .
  • The place-items property is shorthand for align-items and justify-items .
    The place-self property is shorthand for align-self and justify-self .
  • Block Axis(align-item and align-self properties control alignment on the block axis)
  • Inline/Row Axis(justify-items and justify-self properties control alignment on the Inline/Row Axis)
