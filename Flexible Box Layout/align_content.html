justify-content property accepts the same values as align-content.

  • justify-content: flex-start
  • justify-content: flex-end
  • justify-content: center
  • justify-content: space-between
  • justify-content: space-around
  • justify-content: space-evenly (not defined in the Flexbox specification)

flex : 1 1 auto (shorthand of flex-grow, flex-shrink, flex-basis)

  • flex-grow: 3
  • flex-shrink: 1
  • flex-basis: 200px(size of the first item)

flex-flow: row wrap;(横排，串行)

  • flex-direction
    • direction: rtl;(right-to-left)
    • flex-direction: row;(left-to-right)
    • flex-direction: row-reverse;(right-to-left)
    • flex-direction: column;(top-to-bottom)
    • flex-direction: column-reverse;(bottom-to-top)
  • flex-wrap : wrap;(往下自动串行)
